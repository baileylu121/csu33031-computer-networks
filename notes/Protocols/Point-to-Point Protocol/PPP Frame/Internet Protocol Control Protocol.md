_Internet Protocol Control Protocol_ is a communication protocol used in [[Point-to-Point Protocol]] networks to configure and negotiate **IP addresses**.

# Codes
The meaning of different _Internet Protocol Control Protocol_ codes can be interpreted as:

| Code | IPCP              |
| ---- | ----------------- |
| 01   | Configure Request |
| 02   | Configure ACK     |
| 03   | Configure NAK     |
| 04   | Configure Reject  |
| 05   | Terminate Request |
| 06   | Terminate Ack     |
| 07   | Code Reject       |
