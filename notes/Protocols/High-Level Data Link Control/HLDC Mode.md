A _[[High-Level Data Link Control|HLDC]] Mode_ is a control state for [[High-Level Data Link Control|HLDC]] which decides the interactions between [[HLDC Station#Primary Station|primary stations]] and [[HLDC Station#Secondary Station|secondary stations]].


| Mode                             | [[HLDC Station#Primary Station\|Primary]] and [[HLDC Station#Secondary Station\|Secondary]] Role                           | [[Transmission Mode]] |
| -------------------------------- | -------------------------------------------------------------------------------------------------------------------------- | --------------------- |
| Normal Response Mode (NRM)       | [[HLDC Station#Primary Station\|Primary]] controls all communication                                                       | [[Half-Duplex]]       |
| Asynchronous Response Mode (ARM) | [[HLDC Station#Secondary Station\|Secondary]] may transmit at any time                                                     | [[Duplex]]            |
| Asynchronous Balanced Mode (ABM) | Both [[HLDC Station#Primary Station\|Primary]] or [[HLDC Station#Secondary Station\|Secondary]] may transmit at any moment | [[Duplex]]            |
