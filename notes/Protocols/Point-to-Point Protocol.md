_Point-to-Point Protocol_ is a **data link layer** protocol which transfers data directly between two routers with **nothing in between**.

# Life-cycle
The _Life-cycle_ of _Point-to-Point Protocol_ takes the following steps:
1. Negotiate Protocol
2. Authenticate
3. Transmit data
